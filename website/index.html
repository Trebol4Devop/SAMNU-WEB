SAMNU es el software definitivo para estudiantes de ingeniería y ciencias. Más de 30 algoritmos, generación automática de reportes PDF y un parser matemático avanzado.

Potencia tu Aprendizaje

Diseñado para superar las limitaciones del cálculo manual y la falta de herramientas modernas.

+30 Algoritmos

Desde ecuaciones no lineales hasta ecuaciones diferenciales, todo en una sola plataforma.

Reportes PDF

Genera tareas y hojas de trabajo completas con tablas de iteraciones y procedimientos.

Parser Avanzado

Escribe funciones matemáticas de forma natural (ej. "2x + sin(x)") con precisión controlada.

Enfoque Didáctico

Visualiza el paso a paso de cada método para entender la lógica detrás del resultado.

Interfaz Intuitiva

Una experiencia de usuario moderna construida con Flutter.
